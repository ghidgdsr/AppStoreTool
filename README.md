## Instruction（简介）
An app that allows you to search the apps & their details on the App Store via your Android devices.Enhanced by Apple iTunes API.

一个让你在 Android 设备上搜索 App Store 应用程序和对应详情的小工具。由 Apple iTunes API 驱动。
