## Instruction（简介）
An app that allows you to search the apps & their details on the App Store via your Android devices.Enhanced by Apple iTunes API.

一个让你在 Android 设备上搜索 App Store 应用程序和对应详情的工具软件。由 Apple iTunes API 驱动。

## Compile（编译指南）
- Use the command below in your terminal to clone this repo.在命令行中执行以下命令 Clone 本仓库

  ```git clone https://github.com/ghidgdsr/AppStoreTool```
- Import the project into your IDE(like Android Studio)
将项目导入到 IDE（例如 Android Studio）中
